> 第一次,当它本可进取时，却故作谦卑；<br/>
> 第二次,当它空虚时，用爱欲来填充； <br/>
> 第四次,它犯了错，却借由别人也会犯错来宽慰自己；<br/>

> 第六次,当它鄙夷一张丑恶的嘴脸时，却不知那正是自己面具中的一副；<br/>
